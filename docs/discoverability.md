## 中文

这个项目不只是 region selector。更准确的定位是：

- DensePose CSE `vertex_map` 可视化工具
- SMPL / `smpl_27554` vertex ID inspector
- 2D image-to-3D surface mapping 检查器
- 人体部位 region map 编辑器
- 面向预标注 pipeline 的 body surface debugging tool

## Google 收录

Google 和 GitHub search 都不是实时的。README、repository description、topics 和外部链接更新后，
通常还需要一段时间才会被重新抓取。最有效的做法是：

- 保持 README 第一屏包含核心关键词。
- 给 GitHub repo 设置清晰 description 和 topics。
- 在项目 issue、release、blog、论文补充材料、知乎/Medium/Reddit/Hacker News 等地方放真实链接。
- 如果后续有项目主页，可以用 Google Search Console 提交 sitemap；单纯 GitHub repo 通常只能等待抓取。
- 发布第一个 release，让 GitHub 生成更稳定的项目页面和可索引版本节点。
